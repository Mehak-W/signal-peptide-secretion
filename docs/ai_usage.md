# AI / Agent Usage

The author used an AI assistant (Anthropic's Claude) as a tool during the
reproducibility review and revision of this project. As with any tool, the author
directed the work and is responsible for the research questions, methodology,
interpretation, and all claims.

The results do not depend on any tool used: every number is produced by the
committed scripts from the committed data, and is reported as a mean ± standard
deviation to account for GPU nondeterminism.
